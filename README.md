## メモ
- コンストラクタとは、クラスを元にインスタンス化する際に処理を行うメソッドのこと。
- コンストラクタはメソッドだけど、特別な名前（__construct）、呼ばなくてもインスタンス化された時点で動く、初期設定
- コンストラクタの書き方
```
 <?php
 class Product {
  // コンストラクタを定義する
  public function __construct() {
   初期化の内容
  }
 }
 ?>
```
