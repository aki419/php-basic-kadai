## メモ
- 関数とは、一連の処理をひとまとめにして、何度でも再利用できるようにする仕組みのこと
- 関数の作り方
function 関数名() {
    一連の処理
}
Copy

- 関数を作成することを「関数を定義する」という
- プログラミング言語で関数名をつけるときは、「動詞＋目的語」の形をとることが多い
処理の内容	  関数名	    例
○○を追加する	add_○○	add_product（商品を追加する）
○○を削除する	remove_○○	remove_product（商品を削除する）
○○が存在するかどうかチェックする	has_○○	has_product（商品が存在するかどうかチェックする）
○○の状態になっているかどうかチェックする	is_○○	is_purchased（購入されたかどうかチェックする）

- 関数を呼び出す（実行する）には、関数名を記述するだけでよい（例：say_good_morning();）
- 関数は呼び出して初めて実行される

- 関数の()内に引数を記述することを、プログラミングでは「引数を渡す」と表現する
- 引数（ひきすう）とは、関数に与えるデータのことである。
- 関数には複数の引数を渡すこともできる
- 関数内でreturnを記述すれば、戻り値を返せる
- 型宣言とは、関数に渡す引数や返ってくる戻り値のデータ型を指定すること
- PHPファイルの先頭にdeclare(strict_types=1);を記述すれば、型宣言と異なるデータ型の場合にエラーが発生する
