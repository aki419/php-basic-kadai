## メモ
- 連想配列とは、配列におけるインデックスの代わりにキーと呼ばれるラベルをつけて管理するデータのまとまりのことである
- 連想配列は['キー1' => 値1, 'キー2' => 値2, 'キー3' => 値3, ……]のように、[]の中にカンマ区切りで要素（キーと値のセット）を入れていく
- 連想配列の値を取得・更新・追加するには、連想配列名['キー名']のように記述する
