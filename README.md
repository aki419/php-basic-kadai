## メモ
- スコープとは変数を使える範囲（有効範囲）のこと
- PHPのスコープはローカルスコープ（関数内）とグローバルスコープ（関数外）の2つにわけられる
- ローカルスコープで定義された変数をローカル変数、グローバルスコープで定義された変数をグローバル変数という
- 同じ変数名であっても、リーカル変数とグーバル変数は別物として扱われる
- スコープの範囲外で変数を使った場合、エラーが発生する
- クラスとはモノ（オブジェクト）の設計図
- クラスを元にオブジェクトを作ることをインスタンスかという
- クラスの作り方
```
// クラスを定義する
class クラス名 {
  クラスの特徴
}
```
- クラス名は慣習的に先頭を大文字にする
- インスタンス化する方法
```
// インスタンス化する
インスタンスを入れる変数名 = new クラス名();
```
- プロパティの書き方
```
class Product {
// プロパティを定義する
  pubric $name;
}
```
