## メモ
- プロンプト入力時は、マークダウン記法を用いて条件をリスト化すればプロンプトが構造化されてAIが理解しやすくなる。
```
以下の条件を満たすクラスを作成してください
- プログラミング言語:
- クラス名:
- プロパティ
 - 〇〇〇〇
 - ▽▽▽
- コンストラクタ:
- メソッド:
```
- 実行内容の書き方は、番号付きのリスト化すれば実行順序が明確になる
```
以下の実行結果を作成してください
1. 以下のインスタンスを作成する
    1. 〇〇〇〇:
    2. ▽▽▽:
2. 各プロパティに直接アクセスし、値を出力する
3. メソッドを実行し、その結果を出力する
```
