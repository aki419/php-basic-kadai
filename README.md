## メモ
- $i += 2; は次の式と同じ意味
$i = $i + 2;

- while文を使えば、「条件を満たしている間」同じ処理を繰り返し行うことができる
- while文もfor文と同様、無限ループを防ぐために「条件式がFALSEを返す可能性はあるか」を必ず確認する
- while文もfor文と同様、break文やcontinue文が使える
break文：繰り返し処理の途中で強制終了し、ループから抜け出す命令
continue文：繰り返し処理の途中で中断し、次のループに進む命令
break文やcontinue文は、一般的に条件分岐のif文と組み合わせて使う
