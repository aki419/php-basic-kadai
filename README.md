## メモ
- PHP基礎の復習した上で課題のコード作成
- // 関数の作り方
```php
function 関数名() {
    一連の処理
}

// 関数の呼び出し方
関数名();
```
- if文の書き方
```php
if (条件式) {
    条件が成り立つときの処理
}
```

- 等価演算子と厳密等価演算子の違い
・==（等価演算子）：2つの値が等しい場合はTRUEを返す
・===（厳密等価演算子）：2つの値とデータ型が等しい場合はTRUEを返す

-  foreach文の書き方（配列の場合）
```php
foreach ( 配列名 as 変数名）{
  echo 変数名 . '<br>';
}
```

- foreach文の書き方（連想配列の場合）
```php
foreach (配列名 as $key => $value) {
  echo "{$key}は{$value}"
}
```
