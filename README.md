## メモ
- for文は、「決まった回数」同じ処理を繰り返し行いたいときによく使われる
- 条件式が常にTRUEを返す場合、無限ループが発生してしまう
- 無限ループを防ぐために、繰り返し処理を行うときは「条件式がFALSEを返す可能性はあるか」を必ず確認する
- $i += 1や$i -= 1のように単純に1増やしたり1減らしたりしたい場合は、インクリメント演算子++やデクリメント演算子--を使うことで簡潔に記述できる
- break文：繰り返し処理の途中で強制終了し、ループから抜け出す命令
- continue文：繰り返し処理の途中で中断し、次のループに進む命令
- break文やcontinue文は、一般的に条件分岐のif文と組み合わせて使う
